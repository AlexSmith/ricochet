# Ricochet

A dummy endpoint tool for learning Rust and performance testing applications.

## Purpose

Ricochet allows you to:
- Test application performance against a lightweight server

## Getting Started

```bash
cargo run
```

## Usage

localhost:3030/ok

## Features

- Ok endpoint that returns "ok" and unix timestamp example :  ```json {"response":"ok","timestamp":1771350584246}```
